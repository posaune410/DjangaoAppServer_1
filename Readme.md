## システム概要
- 幼稚園や保育園での、児童の出欠管理を行うための簡易的なwebアプリケーションです。
- 保育士は、児童一覧から保護者によって登録された児童の出欠情報を確認することができます。
- 保育士は児童の出欠登録を確認し、欠席の場合には確認ボタンを押すことで保護者に確認された旨が送信されます。
